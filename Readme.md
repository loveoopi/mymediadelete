# Telegram Media Deleter Bot

A Telegram bot that automatically deletes all media messages including photos, videos, stickers, GIFs, and audio.

## Features
- Automatically deletes all media messages
- Works for all users including admins and owners
- Deployable on Heroku

## Deployment on Heroku

1. **Create a new GitHub repository** and push these files

2. **Create Heroku app:**
   ```bash
   heroku create your-app-name
